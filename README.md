# Netflix Sentiment Analysis

## Project Overview

This project performs sentiment analysis on Netflix content descriptions using the VADER Sentiment Analyzer.

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- VADER Sentiment

## Dataset

Netflix Movies and TV Shows Dataset

## Steps Performed

- Loaded the dataset
- Selected title and description columns
- Applied VADER sentiment analysis
- Classified each description as Positive, Neutral, or Negative
- Visualized the sentiment distribution

## Results

- Positive descriptions
- Neutral descriptions
- Negative descriptions

## Conclusion

VADER successfully classified Netflix descriptions into different sentiment categories, providing insights into the overall tone of the content descriptions.
